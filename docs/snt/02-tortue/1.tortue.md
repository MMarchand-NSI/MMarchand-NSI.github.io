# Dessiner avec la tortue

Tu sais afficher du texte et ranger des valeurs dans des variables. Maintenant, on écrit des programmes qui **dessinent**. Pour cela, on va **créer une tortue** qui tient un crayon, la ranger dans une variable, et lui donner des ordres : elle avance, elle tourne, et elle laisse un trait derrière elle.

!!! note "Cours ici, exercices sur Capytale"
    Comme d'habitude, cette page est le **cours**. Les **exercices**, où tu écris et exécutes le code, sont sur **Capytale**.

## Créer la tortue

On commence toujours pareil :

```python
from turtle import Turtle
hortense = Turtle()
```

- `from turtle import Turtle` va chercher le modèle de tortue dans la boîte à outils `turtle`.
- `hortense = Turtle()` **crée** une tortue et la **range dans une variable**. Rappelle-toi : une variable est une boîte. Ici, la boîte contient une tortue.

Le nom `hortense` est un **choix**. Tu peux appeler tes variables comme tu veux (`robert`, `ninja`, `zoe`…) : **c'est toi le maître de ton programme.** Ce qui compte, c'est de parler à ta tortue par ce nom ensuite.

## Lui donner des ordres

La **notation pointée**, c'est tout simplement la façon de **parler à la tortue** : on lui envoie un message. On écrit son nom, un **point**, puis l'ordre qu'on lui donne.

`hortense.forward(100)` se lit « **hortense, avance de 100** ». À gauche du point, à qui on parle (hortense) ; à droite, le message qu'on lui envoie (avance de 100).

| Ordre | Effet |
|---|---|
| `hortense.forward(100)` | avancer de 100 pas, en traçant |
| `hortense.left(90)` | tourner à gauche de 90 degrés |
| `hortense.right(90)` | tourner à droite de 90 degrés |

Les angles sont en **degrés** : `hortense.left(90)` fait un quart de tour.

Un premier programme complet :

```python
from turtle import Turtle
hortense = Turtle()
hortense.forward(100)
```

Il crée la tortue, puis lui fait tracer un trait de 100 pas.

!!! tip "La tortue a une direction"
    Contrairement à la grille de la première séance (droite, haut…), hortense ne se déplace pas en absolu : elle **avance devant elle** et elle **tourne**. Elle regarde toujours quelque part.

## Dessiner un carré

Souviens-toi du carré de la première séance, sur la grille. Avec la tortue, un carré, c'est : avancer, tourner d'un quart de tour, et recommencer quatre fois.

```python
from turtle import Turtle
hortense = Turtle()
hortense.forward(100)
hortense.left(90)
hortense.forward(100)
hortense.left(90)
hortense.forward(100)
hortense.left(90)
hortense.forward(100)
hortense.left(90)
```

## Répéter avec une boucle

On l'a vu dans le langage de tracé : quand ça se répète, on l'écrit **une seule fois**. En Python, c'est la **boucle `for`** :

```python
from turtle import Turtle
hortense = Turtle()
for i in range(4):
    hortense.forward(100)
    hortense.left(90)
```

`for i in range(4):` veut dire « répète 4 fois ». Les instructions **indentées** en dessous forment le bloc répété, exactement comme le bloc indenté après `répète 4 fois :` de la première séance. Ce programme dessine le même carré.

!!! tip "L'indentation, encore"
    Comme dans le langage de tracé, le **décalage vers la droite** indique ce qui est répété. Ce qui n'est pas indenté n'est pas dans la boucle.

## Donner un nom à un dessin : la fonction

Pour dessiner plusieurs carrés, on ne va pas recopier la boucle à chaque fois. On **définit** une fois pour toutes une nouvelle instruction, `carre`, avec le mot-clé `def` :

```python
from turtle import Turtle
hortense = Turtle()

def carre():
    for i in range(4):
        hortense.forward(100)
        hortense.left(90)

carre()
```

- `def carre():` **définit** une nouvelle instruction appelée `carre`. Les lignes indentées en dessous disent ce qu'elle fait.
- `carre()` **appelle** l'instruction : c'est à ce moment que le carré est réellement dessiné.

Une fois `carre` définie, on peut l'appeler autant de fois qu'on veut.

## Ce qu'on retient

- On **crée** une tortue et on la **range dans une variable** (au nom qu'on veut) ; on lui donne des ordres avec un **point** (`hortense.forward(100)`).
- La tortue **avance** (`forward`) et **tourne** (`left`, `right`) : elle a une direction.
- Une **boucle `for`** répète un bloc indenté : `for i in range(4):`.
- Une **fonction** (`def`) donne un nom à une suite d'instructions, pour la réutiliser en l'**appelant**.
