# Des fonctions réglables

Tu sais définir une fonction comme `carre()`. Problème : elle dessine **toujours** un carré de la même taille. Pour un petit carré et un grand carré, il faudrait deux fonctions différentes… ou une seule fonction **réglable**.

!!! note "Cours ici, exercices sur Capytale"
    Cette page est le **cours**. Les **exercices** sont sur **Capytale**.

## Un réglage : le paramètre

On donne à la fonction un **réglage**, qu'on écrit entre les parenthèses : c'est un **paramètre**. Ici, la taille du carré :

```python
from turtle import Turtle, done
hortense = Turtle()

def carre(taille):
    for _ in range(4):
        hortense.forward(taille)
        hortense.left(90)

carre(50)
carre(150)
done()
```

- `def carre(taille):` définit une fonction avec un **paramètre** appelé `taille`.
- À l'intérieur, `taille` se comporte comme une **variable** : elle vaut ce qu'on a mis entre les parenthèses au moment de l'appel.
- `carre(50)` appelle la fonction en donnant `taille = 50` ; `carre(150)` la rappelle avec `taille = 150`.

Une seule fonction, et autant de tailles qu'on veut. C'est tout l'intérêt du paramètre : **une fonction générale plutôt qu'une fonction figée**.

## Se déplacer sans tracer

Sur le dessin ci-dessus, les deux carrés partent du même coin et se chevauchent. Pour les poser **à des endroits différents**, il faut pouvoir déplacer la tortue **sans tracer**, comme le `lève` / `baisse` du langage de tracé de la première séance. En Python :

```python
hortense.penup()      # lever le crayon : les déplacements ne tracent plus
hortense.pendown()    # baisser le crayon : on trace de nouveau
```

Par exemple, un petit carré, un déplacement à vide, puis un grand carré :

```python
from turtle import Turtle, done
hortense = Turtle()

def carre(taille):
    for _ in range(4):
        hortense.forward(taille)
        hortense.left(90)

carre(50)
hortense.penup()
hortense.forward(120)
hortense.pendown()
carre(50)
done()
```

## Ce qu'on retient

- Un **paramètre** est un réglage donné à une fonction entre parenthèses : `def carre(taille):`, puis `carre(50)`.
- À l'intérieur de la fonction, le paramètre se comporte comme une **variable**.
- `penup()` / `pendown()` lèvent et baissent le crayon (comme `lève` / `baisse`), pour déplacer la tortue sans tracer.
- Le cours est ici ; on s'entraîne sur **Capytale**.
